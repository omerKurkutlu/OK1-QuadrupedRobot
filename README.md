# OK1-QuadrupedRobot
Quadruped Robot based on Arduino, also ROS&amp;Gazebo supported...

    1.1)  OK1
    1.2)  RemoteController
2)    CAD-Hardware
3)    ROS_ws
4)    media

## Launch Gantry in ROS
For only Rviz
```bash
roslaunch ok1_moveit_config demo.launch
```
For Gazebo
```bash
roslaunch ok1_moveit_config demo_gazebo.launch
```
